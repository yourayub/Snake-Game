# Snake Game (Python + Pygame)

A modern Snake Game made with Python and Pygame, featuring animated UI, emoji fruits, difficulty levels, pause/resume controls, high-score tracking, and clean game screens.

## Game Details

- **Project Name:** Snake Game  
- **Language:** Python  
- **Main Library:** Pygame  
- **Extra Library:** Pillow (PIL) for image effects  
- **Platform:** Desktop (Windows/Linux/macOS with Python support)
- **Main File:** `snake_game.py`

### Core Features

- Smooth snake movement on a fixed grid
- Emoji-based fruits with subtle animation
- Score and best-score display during gameplay
- High score persistence using local file storage
- Difficulty selection: Easy, Medium, Hard
- Pause/Resume support (`P` key and on-screen button)
- Multiple screens:
  - Home
  - About
  - High Score
  - Contact
  - Difficulty Selection
  - Game Over
- Interactive animated buttons with hover effects
- Contact links for social profiles/email

## How to Run Locally

1. Make sure Python is installed.
2. Install required packages:

```bash
pip install pygame pillow
```

3. Run the game from the project folder:

```bash
python snake_game.py
```
4. python snake_game.py


## Controls

- **Arrow Keys / WASD:** Move snake
- **P:** Pause/Resume
- **Mouse:** Click UI buttons and menu options

## Project Files (Important)

- `snake_game.py` - Main game source code
- `highscore.txt` - Stores the highest score locally
- `scores.txt` - Stores recent scores (if generated)
- `cover.png` - Game visual asset (if used by your design)
